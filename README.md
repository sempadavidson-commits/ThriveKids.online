# ThriveKids Global Foundation

## About Us

ThriveKids is a leading international nonprofit organization dedicated to empowering children and families through accessible education, comprehensive health support, child protection, and community development initiatives.

Our mission is to create lasting positive change in the lives of vulnerable children worldwide by providing essential resources and support systems.

## Vision & Mission

**Vision:** A world where every child has the opportunity to thrive, learn, and reach their full potential.

**Mission:** To deliver holistic support services to children and families in underserved communities through education, healthcare, protection, and community empowerment programs.

## Our Core Programs

- **Education:** Providing access to quality education and digital learning resources
- **Health & Wellness:** Supporting preventive healthcare and health education
- **Child Protection:** Ensuring safety and welfare of vulnerable children
- **Community Development:** Strengthening local communities through sustainable initiatives

## Getting Started

This repository contains the codebase for our digital platform.

**Prerequisites:** Node.js 18+

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env.local
   ```
   Add your API keys to `.env.local`

### Development

Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`

### Build

Create a production build:
```bash
npm run build
```
